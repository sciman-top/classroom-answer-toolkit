# 试卷参考答案交付规范-初中数学-完整版 v0.1

> 生成方式：自动汇编生成，禁止手改。
> 目标 subject-pack：`math-answer`。
> 汇编清单：`prompts/specs/assemblies/math.json`。
> 产物类型：完整版。
> 汇编层：平台总则(v1.0)、K12通用(v1.0)、图文定量题通用(v1.0)、初中数学特异(v0.1)。
> 输出版本：v0.1。

---
## 平台总则

> 适用范围：面向中小学、多学科的试卷参考答案生成、校验、渲染与交付平台规则。
> 文件角色：本文件定义平台共性约束；具体学段与学科规则由 `commons/` 和 `subjects/` 补充。

## 1. 分层约定

1. 平台总则负责通用交付约束、目录约定、文件命名、渲染验收和资产同步规则。
2. `commons/` 负责跨多个学科或多个学段复用的规则。
3. `subjects/` 负责学段/学科专用源规范。
4. `compiled/` 负责对外转发和工具 `humanSpec` 所引用的完整规范产物。

## 2. 目录与命名

1. `样例交付/` 用于回归验证、演示和工具链示例。
2. `正式交付/` 用于真实生产交付。
3. `prompts/specs/` 是人类规范真值区，`docs/strategy/` 是规划与执行真值区。
4. 平台文件命名：`试卷参考答案交付规范-平台总则-v<version>.md`
5. 学段学科文件命名：`试卷参考答案交付规范-<学段学科>-源规范-v<version>.md`
6. compiled 文件命名：`试卷参考答案交付规范-<学段学科>-完整版-v<version>.md`

## 3. 交付基线

1. 默认最终交付物至少包含答案 `.md` 与答案 `.pdf`。
2. PDF 中数学公式必须走真实 LaTeX 数学渲染链。
3. 任何交付都应可追踪到使用的 subject-pack、snapshot 与人类规范版本。
4. 任何命名、目录或规范版本调整，都必须同步更新工具配置、测试样例和文档引用。
5. 默认主交付链不承诺 AI 自动基于题图生成答案图；若需插图，只支持用户提供或人工复核后的受控图块插入。

## 4. 人工复核与风险边界

1. 对高风险视觉内容，平台必须允许人工复核、抽检与降级。
2. 工具输出不得暴露内部审查过程、试错痕迹或模型自评。
3. 若证据不足，应标【疑】或进入复核，不得硬猜。
4. 若只是局部题图模糊或局部视觉理解不确定，但不阻断整份交付，应继续完成其余可确定部分，并在对应小问标注【疑】/待复核。
5. 只有当不确定性会阻断大范围求解、影响整份交付主结论，或用户明确要求实时确认时，才应暂停并提问。

## K12通用

> 适用范围：面向中小学各学段、各学科的试卷参考答案生成、校验、渲染与交付。
> 文件角色：承载跨学科、跨学段都普遍适用的规范。

## 1. 证据优先级

1. 原始 PDF、原始图片、原始截图是最高依据。
2. OCR、自动解析文本、上一轮答案、其他模型答案只能作为辅助。
3. 若原图与 OCR 冲突，以原图为准。
4. 若原图模糊、关键数据缺失、图表不可辨认，不得硬猜，必须标注【疑】。
5. 若疑点只影响局部小问，默认继续完成其他可确定部分，不因单点疑问中断整份交付。

## 2. 交付约束

1. 默认最终交付物至少包含答案 `.md` 与答案 `.pdf`。
2. PDF 中数学公式必须走真实 LaTeX 数学渲染链，不得降级为纯文本近似。
3. 最终输出不得暴露内部审查过程、试错痕迹、工具调用信息或模型自评。
4. 若存在疑点，应输出极简疑点清单，而不是在答案正文中混入长说明。
5. 若需要附图，默认仅允许插入用户提供或人工复核后的图块，不把 AI 自动看图作图作为通用承诺。

## 3. 目录与命名

1. `样例交付/` 用于演示、回归、冒烟测试，不与正式交付混放。
2. `正式交付/` 用于真实生产交付。
3. 人类规范采用“源规范 + compiled 生成物”模式，compiled 文件用于转发与工具引用。

## 4. 可读性与答题风格

1. 答案应优先服务课堂展示、打印分发与复核，不应写成冗长讲解稿。
2. 应优先输出结论、关键式、必要前提，不堆叠无新增价值的解释句。
3. 若用户明确要求补分析，应只补与结论直接相关的必要推导。

## 5. 人工复核原则

1. 高风险内容应允许进入人工复核，而不是强行全自动放行。
2. 复核结论应可追踪到具体证据，而不是停留在主观描述。
3. 仅当疑点会阻断整份交付的主要判断，或用户明确要求即时确认时，才暂停提问。

## 图文定量题通用

> 适用范围：适用于多数学科中的图像、表格、坐标图、仪表、实验图、结构图、定量读图题。
> 文件角色：承载跨学科图文定量题的共性约束。

## 1. 图文证据优先

1. 题目若要求“根据图”“由表可知”“读数可得”，必须先抽取图文证据，再进入求解。
2. 同一道题有多幅图或多组数据时，必须先完成“小问 -> 图号/数据源”绑定，不得跨图误答。
3. 计算只作一致性校验，不得替代应由图表直接读出的答案。
4. 高风险视觉题不得只引用整页图；必须尽量形成 `questionRef -> figureRef -> cropRef -> evidenceRef` 的证据链。
5. 若当前链路只能得到 OCR 文本或整页截图，不能静默假装已完成看图；应按对应小问标【疑】/待复核。

## 2. 视觉证据编译器

1. 输入页应先归一化为 `NormalizedPage`，记录页图、页号、DPI、方向、deskew、裁边、去噪、多尺度渲染和图像质量标记。
2. 题区、图区、表格区、公式区、坐标轴区、刻度区、图例区和局部 crop 应记录为 `VisualRegion`。
3. 每个图文相关小问应聚合为 `ProblemEvidenceBundle`，只引用页、区域、图块、OCR、layout、图元和 crop 的 id，不复制视觉对象。
4. 每条求解或校验轨道输出 `TrackResult`，记录候选答案、证据引用、冲突、缺失证据、置信度和风险。
5. 最终放行、标疑或复核进入 `DecisionRecord`；`visualReviewPassed=null` 表示未裁定或自动降级，`trusted=false` 直到无未决题且 review 生命周期批准。

## 3. 仪表与刻度读数

1. 仪表读数必须明确读数对象、量程或零点/起点、分度值、相对位置和最终数值。
2. 若读数差异发生在最小分度级别，必须重新放大原图，从相邻主刻度两个方向复读。
3. 不得把阴影、外框、装饰线、黑块、游码整体宽度等非读数要素误当读数依据。

## 4. 视觉高风险分类

以下题型默认视为视觉高风险：

- 刻度/仪表读数题
- 几何关系或图像关系题
- 坐标图/函数图读图题
- 实验装置图与电路图
- 作图题
- 多图多问绑定题
- 模糊、倾斜、遮挡、低清晰度题图
- 题图与 OCR 冲突、题图与参考答案冲突

## 5. 三轨与复核

1. 轨道 A：多模态视觉直答。
2. 轨道 B：OCR、图元抽取、结构化证据后再求解。
3. 轨道 C（Track C）：规则校验器，检查单位、量程、分度值、坐标轴、图号绑定、答案格式和学科约束。
4. 任一轨道冲突、证据不足、绑定不稳、低置信或规则校验阻断，不能视为可信。
5. 双轨一致但证据链缺失、低置信或命中高风险分类时，不得直接视为可信。
6. 必须保留哨兵样例与抽检机制，防止双轨一致同错。
7. 题图模糊、遮挡或读图不稳时，默认策略是先完成可确定部分，并在对应小问标【疑】/待复核；不是立即中断整题或整卷。
8. 只有当图像疑点会同时影响多个关键小问、无法建立最基本的图号绑定或量纲边界时，才暂停提问。

## 6. 较难题分析边界

1. 视觉题若只是看图读数出错，不默认展开成长分析。
2. 只有关系隐蔽、分段明显、阈值判断、图像综合统筹类小问，才属于应补充分析的较难题。

## 初中数学特异

> 适用范围：初中数学试卷参考答案生成、校验、渲染与交付。
> 文件角色：初中数学实验 subject-pack 的学科特异性源规范。
> 依赖层：平台总则、K12通用。
> 说明：本文件用于把 `math-answer` 纳入统一 human-spec 汇编链；当前只固定共享渲染/校验 contract 与少量数学格式约束，不宣称完整题型覆盖。

## 1. 当前范围

1. 当前 pack 的主要目标是验证平台 contract、snapshot、validator 与交付链路不依赖物理专有规则。
2. 当前自动化覆盖包括基础代数 smoke、分步等价变形 smoke、基础统计结论 smoke、基础概率记号 smoke、函数图像 review-fallback smoke、几何 review-fallback smoke、图表驱动统计/概率 review-fallback smoke、公式定界符硬失败和跨学科资产校验。
3. 函数图像读图、几何作图与图表驱动的统计/概率题目前只覆盖“无法稳定取证时输出【疑】/待复核”的降级交付，不得把当前 pack 视为生产级初中数学能力。

## 2. 数学特异性约束

1. 需要推导时，优先保留课堂友好的分步等价变形，不要压缩成只剩最终结果的一跳答案。
2. 公式、方程、分式、根号、上下标和单位必须走真实 LaTeX 渲染，不得把源码痕迹直接交付。
3. 对文本即可确定的数据列表基础统计题，可直接给出平均数、中位数、众数等标准结论，必要时可保留简洁计算式以便课堂展示。
4. 对文本即可确定样本空间与事件定义的基础概率题，可使用 `$P(A)=\frac{m}{n}$` 等标准记号，并保持事件名、分子分母含义和最终结论清晰。
5. 对依赖函数图像、坐标读图或几何构造的高风险小问，若当前规则不足以稳定表达证据链，应在对应小问输出紧凑的 `【疑：...待复核】` 标记，而不是硬猜答案。
6. 对依赖统计图表、坐标图表或图表统计口径判断的高风险小问，若当前规则不足以稳定表达证据链，应在对应小问输出紧凑的 `【疑：图表统计口径待复核】` 或等价 review-fallback 标记。
7. 若题目依赖图形、坐标图、统计图表、实验过程或几何构造，而当前规则不足以稳定表达证据链，应进入人工复核或降级说明，不得伪装成已覆盖的通用能力。

## 3. 后续补强方向

1. 代数计算与方程组
2. 几何证明、几何作图与几何关系判定
3. 函数图像与坐标系读图
4. 图表驱动的统计、复杂概率与应用题
