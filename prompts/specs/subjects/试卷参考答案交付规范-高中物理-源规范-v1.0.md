# 试卷参考答案交付规范-高中物理-源规范 v1.0

> 适用范围：高中物理试卷参考答案生成、校验、渲染与交付。
> 文件角色：高中物理学段特异性源规范。
> 依赖层：平台总则、K12通用、图文定量题通用、物理通用。
> 说明：本文件是高中物理的首版占位源规范，先固定边界与扩展方向，后续按题型与知识模块持续补足。

## 1. 学段边界

1. 允许比初中更完整的推导与建模过程，但仍应保持课堂可读性，不写成竞赛讲义。
2. 对向量、函数图像、复杂电路、复合场景和多阶段能量过程，应允许保留必要推导链。
3. 涉及高中独有符号、模型或公式时，必须确保答案口径与高中教材和常见考试评分习惯一致。

## 2. 特异性高风险点

1. 图像分段、阈值判定、临界条件和多过程统筹题更容易出现“结论对、证据链错”的情况。
2. 坐标图、函数图、矢量分解、电学实验图和综合实验装置图应优先进入高风险视觉复核。
3. 若题目结论依赖图像交点、斜率、面积或分段读图，应优先沉淀结构化图像证据。

## 3. 后续补强方向

1. 模块一：运动与力学综合
2. 模块二：电磁学与实验
3. 模块三：热学、光学、原子物理
4. 模块四：图像题、实验设计题、综合压轴题
