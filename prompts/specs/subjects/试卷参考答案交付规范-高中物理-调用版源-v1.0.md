# 试卷参考答案交付规范-高中物理-调用版源 v1.0

> 适用场景：日常调用的高中物理参考答案生成、核查、Markdown/LaTeX 排版与 PDF 渲染交付。
> 权威来源：同体系下的高中物理完整版。
> 使用原则：本文件只保留高频入口和关键硬约束，规则冲突时以高中物理完整版为准。

## 1. 高优先级约束

1. 原卷图文是最高依据；OCR 与旧答案只能辅助。
2. 高风险图题默认走视觉证据优先与双轨比对。
3. 计算题保留必要推导，但不得写成长篇讲解稿。
4. 公式必须真实 LaTeX 渲染，PDF 必须可审阅、可打印、可回放。

## 2. 高中物理特异提醒

1. 分段图像、临界条件、矢量关系、复杂电路和综合实验题，允许保留完整但不过度详细的推导。
2. 图像读图和实验装置题一旦证据不足，应标【疑】或进入复核，不得硬猜。
