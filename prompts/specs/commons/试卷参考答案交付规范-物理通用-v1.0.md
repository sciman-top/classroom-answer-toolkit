# 试卷参考答案交付规范-物理通用 v1.0

> 适用范围：跨初高中物理都成立的规则与表达口径。
> 文件角色：承载不依赖学段边界的物理共性规范。

## 1. 物理量与公式

1. 题目问什么物理量，公式等号左边就写什么物理量。
2. 计算题应满足“公式 -> 带单位代入 -> 结果”闭合。
3. 不得混用不同物理量的口径，例如电功、电热、吸热、放热应按题意严格区分。

## 2. 单位与表达

1. 代入式中的题给量默认应带单位。
2. 结果必须带单位，除非题目明确要求只写数值或只写表达式。
3. 题干要求用给定符号表示时，答案只能使用题干给定或直接定义的符号。

## 3. 实验与作图

1. 实验题优先使用得分点句式，不写成长段讲解。
2. 若没有实际画图，不得只写“如图所示”，必须改成精确文字说明。
3. 作图题和实验装置题应优先基于原图叠加、定位和复核，而不是凭空重画。

## 4. 视觉风险防线

1. 电路图、实验装置图、受力图、光路图等题型默认允许进入高风险复核流程。
2. 图读数和图推理应尽量依赖结构化证据，而不是只依赖模型主观视觉判断。
