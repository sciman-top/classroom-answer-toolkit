# 试卷参考答案交付规范-图文定量题通用 v1.0

> 适用范围：适用于多数学科中的图像、表格、坐标图、仪表、实验图、结构图、定量读图题。
> 文件角色：承载跨学科图文定量题的共性约束。

## 1. 图文证据优先

1. 题目若要求“根据图”“由表可知”“读数可得”，必须先抽取图文证据，再进入求解。
2. 同一道题有多幅图或多组数据时，必须先完成“小问 -> 图号/数据源”绑定，不得跨图误答。
3. 计算只作一致性校验，不得替代应由图表直接读出的答案。

## 2. 仪表与刻度读数

1. 仪表读数必须明确读数对象、量程或零点/起点、分度值、相对位置和最终数值。
2. 若读数差异发生在最小分度级别，必须重新放大原图，从相邻主刻度两个方向复读。
3. 不得把阴影、外框、装饰线、黑块、游码整体宽度等非读数要素误当读数依据。

## 3. 视觉高风险分类

以下题型默认视为视觉高风险：

- 刻度/仪表读数题
- 几何关系或图像关系题
- 坐标图/函数图读图题
- 实验装置图与电路图
- 作图题
- 多图多问绑定题
- 模糊、倾斜、遮挡、低清晰度题图

## 4. 双轨与复核

1. 轨道 A：多模态视觉直答。
2. 轨道 B：OCR、图元抽取、结构化证据后再求解。
3. 双轨答案不一致时必须进入人工复核。
4. 双轨一致但证据链缺失、低置信或命中高风险分类时，不得直接视为可信。
5. 必须保留哨兵样例与抽检机制，防止双轨一致同错。
6. 题图模糊、遮挡或读图不稳时，默认策略是先完成可确定部分，并在对应小问标【疑】/待复核；不是立即中断整题或整卷。
7. 只有当图像疑点会同时影响多个关键小问、无法建立最基本的图号绑定或量纲边界时，才暂停提问。

## 5. 较难题分析边界

1. 视觉题若只是看图读数出错，不默认展开成长分析。
2. 只有关系隐蔽、分段明显、阈值判断、图像综合统筹类小问，才属于应补充分析的较难题。
